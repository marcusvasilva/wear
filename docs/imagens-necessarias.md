## Dicas para producao

- **Fotos de produto (configurador):** fundo transparente PNG, iluminacao uniforme, angulo frontal levemente inclinado
- **Fotos ambientadas (hero/prova social):** luz natural, contexto real de uso, alta resolucao (min 1200px largura)
- **Video fabrica:** horizontal (16:9), boa iluminacao, sem audio obrigatorio (pode ter legenda)
- **Todas as imagens web:** otimizar para web (WebP ou JPEG qualidade 80-85%), manter PNG apenas para fundo transparente
